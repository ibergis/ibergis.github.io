5. Crear un esquema de IberGIS de ejemplo

  • Crear un GeoPackage de ejemplo

  • Crear un proyecto QGIS de ejemplo

5.1. Crear un GeoPackage de ejemplo

Para crear un GeoPackage de ejemplo para empezar a trabajar, haga clic en el botón IberGIS y rellene los siguientes campos:

  • Nombre GPKG: nombre del archivo GeoPackage.

  • Descripción: una breve descripción del proyecto.

  • Ruta: ruta donde se almacenará el GeoPackage.

  • Idioma: idioma del GeoPackage. Para el ejemplo es obligatorio Inglés.

  • Filtrar SRID: sistema de referencia de coordenadas del GeoPackage. Para el ejemplo es obligatorio 25831.

  • Datos de ejemplo: opción para crear una plantilla.

Después de especificar lo anterior, si hacemos clic en el botón Aceptar se creará el GeoPackage de ejemplo.

5.2. Crear un proyecto QGIS de ejemplo

Después de crear el GeoPackage veremos que tenemos disponible la pestaña Crear proyecto QGIS donde debemos especificar:

  • Archivo GPKG: nombre y carpeta para el GeoPackage.

  • Nombre de archivo GIS: nombre del proyecto QGIS.

  • Carpeta GIS: carpeta del proyecto.

Al hacer clic en el botón Aceptar, se creará el proyecto de ejemplo.

Ahora estamos listos para empezar a trabajar con el ejemplo.
