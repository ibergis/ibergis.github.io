8.2.5. Diálogo Resultados

  • Resumen de informe

  • Gráfico de series temporales

  • Cargar resultados ráster

  • Cargar resultados vectoriales

  • Establecer carpeta de resultados

Herramienta para trabajar con los resultados de la simulación.

Podemos gestionar perfil, resumen de informe, gráfico de series temporales, cargar resultados ráster, cargar resultados vectoriales y establecer carpeta de resultados.

8.2.5.1. Perfil

Dibuja el perfil, la sección longitudinal entre dos o más nodos.

Las siguientes opciones están disponibles:

  • Perfil: permite seleccionar los nodos inicial y final del perfil.

  • Parámetros: permite seleccionar parámetros como modo o desplazamientos. Las opciones disponibles son:

    • Modo: permite seleccionar el modo del perfil. Los modos disponibles son: Seco y Con agua.

    • Desplazamientos: permite seleccionar los desplazamientos del perfil. Los desplazamientos disponibles son: Profundidad y Elevación.

    • Marca de tiempo: permite seleccionar la marca de tiempo del perfil. Existe la opción de seleccionar un instante o un período.

Importante

Las opciones de Marca de tiempo solo funcionarán en modo Con agua.

Después de hacer clic en el botón Dibujar perfil, se abrirá un nuevo diálogo con el perfil.

Este diálogo tiene muchas opciones para mejorar la visualización del perfil, como zoom o personalización de los colores.

Nota

El modo Seco obtiene los datos del GeoPackage del proyecto, mientras que el modo Con agua obtiene los datos de la carpeta de resultados.

8.2.5.2. Resumen de informe

Proporciona los valores máximos de los resultados, en una tabla.

8.2.5.3. Gráfico de series temporales

Permite visualizar cómo varían el flujo, la precipitación o los niveles de agua en puntos específicos del modelo.

Las siguientes opciones están disponibles:

  • Períodos de Tiempo: permite seleccionar los períodos de tiempo a mostrar.

  • Configuración del Gráfico: permite configurar la configuración del gráfico.

  • Series de Datos: permite añadir, eliminar y seleccionar los objetos a mostrar.

  • Selección de Series de Datos: permite seleccionar los objetos del lienzo y su variable a mostrar.

Nota

Esta herramienta permite guardar y cargar la configuración del gráfico con los botones Guardar y Cargar.

Después de hacer clic en el botón Mostrar gráfico, se abrirá un nuevo diálogo con el gráfico.

Este diálogo tiene muchas opciones para mejorar la visualización del gráfico, como zoom o personalización de los colores.

8.2.5.4. Cargar resultados ráster

Permite cargar resultados de simulación en formato ráster.

8.2.5.5. Cargar resultados vectoriales

Carga los valores máximos obtenidos por el modelo 1D, en formato de capa con un estilo específico para cada capa.

8.2.5.6. Establecer carpeta de resultados

Establece la carpeta donde se almacenan los resultados.
