8.2.3. Diálogo Gestor de mallas

  • Diálogo Crear Malla

Herramienta para incorporar la malla al modelo. Tener una malla no solo es esencial para que se realicen los cálculos, ya que define los elementos de cálculo; sino que también es la que obtiene los valores de los parámetros distribuidos del modelo a través de su red, como usos del suelo o número de curva para el modelo de infiltración. La malla puede crearse o importarse directamente. Los diferentes escenarios creados para diversas condiciones de contorno también deben guardarse en una malla existente.

Los siguientes botones están disponibles:

  • Crear: permite crear una nueva malla (ver Diálogo Crear Malla).

  • Ver: permite, al seleccionar una malla, verla en el ToC.

  • Importar: permite importar una malla desde archivos del ordenador.

  • Exportar: permite exportar una malla.

  • Eliminar: permite, al seleccionar una malla, eliminarla.

8.2.3.1. Diálogo Crear Malla

Herramienta para crear una nueva malla. Las siguientes opciones están disponibles:

  • Datos de entrada: comprobaciones a realizar en las capas para detectar errores y permite usar solo las entidades seleccionadas.

  • Transiciones: permite al usuario configurar los ajustes de transición de malla, que controlan qué tan suavemente disminuye la resolución de la malla desde las líneas de quiebre hacia el interior del dominio. Para este propósito, la opción Habilitar transiciones ajustadas debe estar activa. En este caso, el usuario puede definir el valor de la pendiente de transición, la distancia en la que comienza la transición y la extensión de la transición.

  • Condiciones de contorno: permite aplicar las condiciones de contorno del escenario actual a la malla.

  • Elevación: capa DEM para obtener elevaciones.

  • Rugosidad: capa para obtener valores de rugosidad (coeficiente de Manning).

  • Pérdidas: capa para obtener valores de pérdida (Número de Curva).

  • Opciones de malla: este campo permite guardar la malla con un nombre específico. Este nombre le ayudará posteriormente a identificar la malla con la que desea trabajar al asignar el escenario de condiciones de contorno.
