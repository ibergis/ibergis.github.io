8.1.3. Diálogo Crear condiciones de contorno

Herramienta para establecer condiciones de contorno. Al hacer clic en el botón podemos hacer clic en las líneas correspondientes para establecer las condiciones de contorno. Luego, aparecerá un formulario donde debemos especificar:

  • ID: identificador de la condición de contorno.

  • Código: código de la condición de contorno.

  • Descripción: una breve descripción de la condición de contorno.

  • Escenario BC: escenario de condición de contorno en el que se almacenará la nueva condición de contorno.

  • Tipo de Contorno: podemos seleccionar dos opciones:

    • Entrada 2D: la usamos para especificar un flujo de entrada. Es necesario seleccionar una opción para la entrada (Descarga Total o Elevación de Agua) y la condición de entrada (Crítico/Subcrítico).

    • Salida 2D: la usamos para especificar un flujo de salida. Debe seleccionar la condición de flujo, ya sea Subcrítico o Supercrítico/Crítico. - Si se selecciona Subcrítico, también debe definir el tipo de salida (Vertedero o Nivel dado) y la definición de cresta (Altura o Elevación), junto con sus valores correspondientes. - Si se selecciona Supercrítico/Crítico, no se requieren parámetros adicionales, simplemente permite que el flujo generado salga del dominio del modelo.

  • Series Temporales: esta opción nos permite especificar un hidrograma de entrada. Solo está disponible cuando seleccionamos la opción de entrada 2D.

Importante

Las condiciones de contorno se aplican directamente al escenario actual. El escenario actual puede establecerse en Diálogo Gestor de condiciones de contorno.

Una vez que existen las condiciones de contorno, pueden editarse directamente habilitando la edición a través del botón Activar edición, ya sea desde el mapa o la tabla de atributos.
