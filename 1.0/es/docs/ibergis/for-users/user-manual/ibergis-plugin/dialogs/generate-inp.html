8.2.2. Diálogo Generar INP

Herramienta que permite exportar la red a un archivo inp.

Al hacer clic en el botón Opciones se abrirá una ventana donde podemos configurar las opciones de exportación inp.

Luego debemos especificar un nombre y una ruta para el archivo haciendo clic en el botón con tres puntos.
