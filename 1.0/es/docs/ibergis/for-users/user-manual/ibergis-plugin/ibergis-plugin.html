8. Plugin de IberGIS

  • Barra de herramientas Principal

  • Barra de herramientas Utilidades

El plugin de IberGIS es una herramienta esencial para la gestión integrada del ciclo del agua.

Tiene dos barras de herramientas: Principal y Utilidades.

8.1. Barra de herramientas Principal

Esta barra de herramientas contiene los siguientes botones:

  • Importar INP: permite importar un archivo inp. Para más detalles, consulte la sección Diálogo Importar INP.

  • Gestor de condiciones de contorno: permite gestionar las condiciones de contorno. Para más detalles, consulte la sección Diálogo Gestor de condiciones de contorno.

  • Crear condición de contorno: permite crear condiciones de contorno. Para más detalles, consulte la sección Diálogo Crear condiciones de contorno.

  • Gestor de objetos no visuales: permite gestionar los objetos no visuales del proyecto. Para más detalles, consulte la sección Diálogo Objetos no visuales.

  • Acciones de puente: permite añadir o editar un puente. Para más detalles, consulte la sección Diálogo Acciones de Puente.

8.2. Barra de herramientas Utilidades

Esta barra de herramientas contiene los siguientes botones:

  • Opciones: permite gestionar todas las opciones de SWMM e Iber en un solo lugar. Para más detalles, consulte la sección Diálogo Opciones Go2Iber.

  • Generar INP: permite exportar la red a un archivo inp. Para más detalles, consulte la sección Diálogo Generar INP.

  • Gestor de mallas: permite gestionar las mallas del proyecto. Para más detalles, consulte la sección Diálogo Gestor de mallas.

  • Ejecutar modelo: permite ejecutar el modelo para simular la red. Para más detalles, consulte la sección Diálogo Ejecutar modelo.

  • Resultados: contiene muchas herramientas para trabajar con los resultados de la simulación. Para más detalles, consulte la sección Diálogo Resultados.

  • Verificar proyecto: permite verificar el proyecto para encontrar errores. Para más detalles, consulte la sección Diálogo Verificar proyecto.
