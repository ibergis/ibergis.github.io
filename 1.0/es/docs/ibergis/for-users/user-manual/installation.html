3. Instalación

  • Instalación de QGIS

  • Instalación de IberGIS

  • Instalación de SWMM (opcional)

  • Instalación de Iber (opcional)

Como acabamos de ver en la sección ¿Qué es IberGIS?, para trabajar con IberGIS necesitamos tener QGIS instalado.

Aunque no es obligatorio, se recomienda tener instalados SWMM e Iber.

Nota

Para descargar el software a utilizar, se recomienda hacerlo desde las páginas oficiales de cada uno de ellos.

3.1. Instalación de QGIS

QGIS es el cliente GIS que usamos para trabajar con IberGIS.

Para descargarlo iremos al centro de descargas del sitio web de QGIS y cerraremos la ventana de donación.

QGIS está disponible para muchos sistemas operativos diferentes, pero IberGIS solo funcionará completamente en Windows y Linux, por lo que tendremos que seleccionar uno u otro dependiendo de nuestro sistema operativo. Una vez localizado, podemos descargarlo.

Una vez descargado QGIS, ejecute el archivo descargado y siga las instrucciones para instalarlo.

Importante

IberGIS requiere la última versión LTR disponible de QGIS.

3.3. Instalación de SWMM (opcional)

SWMM es un programa desarrollado por la Agencia de Protección Ambiental (EPA) de los Estados Unidos que permite simular las redes de alcantarillado y drenaje urbano en áreas urbanas.

Para descargarlo iremos al sitio web oficial donde desplazaremos por la página hasta localizar el archivo que debemos descargar para poder instalarlo.

Una vez descargado SWMM, ejecute el archivo descargado y siga las instrucciones para instalarlo.

3.4. Instalación de Iber (opcional)

Iber es un modelo hidráulico bidimensional para la simulación de flujo de superficie libre distribuido gratuitamente en inglés y español.

Para descargarlo debemos estar registrados en el sitio web oficial.

Una vez descargado Iber, ejecute el archivo descargado y siga las instrucciones para instalarlo.
