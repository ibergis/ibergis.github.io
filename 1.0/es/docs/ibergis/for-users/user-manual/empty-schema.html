4. Crear un esquema de IberGIS vacío

  • Crear un GeoPackage vacío

  • Crear un proyecto QGIS vacío

4.1. Crear un GeoPackage vacío

Para crear un GeoPackage vacío para empezar a trabajar, haga clic en el botón IberGIS y rellene los siguientes campos:

  • Nombre GPKG: nombre del archivo GeoPackage.

  • Descripción: una breve descripción del proyecto.

  • Ruta: ruta donde se almacenará el GeoPackage.

  • Idioma: idioma del GeoPackage.

  • Filtrar SRID: sistema de referencia de coordenadas del GeoPackage.

  • Datos vacíos: opción para crear un GeoPackage vacío.

Después de especificar lo anterior, si hacemos clic en el botón Aceptar se creará el GeoPackage vacío.

4.2. Crear un proyecto QGIS vacío

Después de crear el GeoPackage veremos que tenemos disponible la pestaña Crear proyecto QGIS donde debemos especificar:

  • Archivo GPKG: nombre y carpeta para el GeoPackage.

  • Nombre de archivo GIS: nombre del proyecto QGIS.

  • Carpeta GIS: carpeta del proyecto.

Al hacer clic en el botón Aceptar, se creará el proyecto vacío.

Ahora estamos listos para importar el archivo inp con la red o comenzar a digitalizarla.
