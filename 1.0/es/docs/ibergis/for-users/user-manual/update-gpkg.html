6. Actualizar un GeoPackage de IberGIS

Herramienta para actualizar un GeoPackage de IberGIS a una nueva versión.

Al hacer clic en el botón con tres puntos, abrimos un explorador para buscar el archivo GeoPackage.

Si el archivo GeoPackage es válido, su versión se mostrará en el campo Versión del proyecto. Cualquier diferencia entre la versión actual del proyecto y la nueva versión se listará en el campo Descripción de la versión.

Después de seleccionar el archivo GeoPackage, podemos hacer clic en el botón Aceptar para actualizar el GeoPackage.

El GeoPackage se actualizará con los nuevos datos.

Importante

Esta herramienta solo soporta actualizaciones de una versión inferior a una versión superior. La versión mínima soportada es 1.0.0.
