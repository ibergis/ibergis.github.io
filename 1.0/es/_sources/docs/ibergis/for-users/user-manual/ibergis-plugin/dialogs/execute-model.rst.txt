.. _dialog-execute-model:

====================
Dialog Execute model
====================

.. only:: html

    .. contents::
       :local:

.. figure:: img/execute-model.png

    Execute model.

Tool to run the simulations. We must specify the mesh we want to use and the folder to save it.
By clicking the *Options* button, we can access the *Go2Iber-options* panel, where we can configure any remaining simulation settings.
By checking the *Generate COG* check box, additional *Cloud Optimized GeoTIFF* files will be generated.
The *Info Log* tab will show the steps the process is performing in real time and all the information about the results.
