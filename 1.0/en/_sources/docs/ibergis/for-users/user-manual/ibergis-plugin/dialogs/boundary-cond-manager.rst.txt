==================================
Dialog Boundary conditions manager
==================================

.. only:: html

    .. contents::
       :local:

Tool to manage the boundary conditions of the project.

.. figure:: img/boundary-cond-manager.png

  Window to manage the boundary conditions.

The following buttons are available:

- Current scenario: allows, by selecting a scenario, to set it as the current one. All the new boundary conditions established will be stored in the current scenario.
  The name of the current scenario will be displayed to the left of this button.
- Create: allows to create a new scenario of boundary conditions. We must specify the *idval*, the *Name* and, optionally, a sort *Description* of the new scenario.
- Duplicate: allows, by selecting a scenario, to duplicate it. We must specify the *idval*, the *Name* and, optionally, a sort *Description* of the new scenario.
- Edit: allows, by selecting a scenario, edit it.
- Delete: allows, by selecting a scenario, delete it.
- Save to mesh: allows, by selecting a scenario, save the boundary conditions in the mesh. **To use this button, at least one mesh must have been created**.
