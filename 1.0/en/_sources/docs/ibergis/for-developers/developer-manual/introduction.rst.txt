============
Introduction
============

.. only:: html

   .. contents::
      :local:

Welcome to the developer's guide for IberGIS, the first free software which aims to perform 1D-2D simulations by combining the SWMM and Iber calculation engines.

This guide is intended for developers who want to contribute to the development of IberGIS.

.. note:: This guide is a work in progress and will be updated as the software evolves.
