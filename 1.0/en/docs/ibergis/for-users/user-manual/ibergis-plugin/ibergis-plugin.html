7. IberGIS Plugin

  • Toolbar Main

  • Toolbar Utilities

The IberGIS plugin is an essential tool for integrated water cycle management.

It has two toolbars: Main and Utilities.

7.1. Toolbar Main

This toolbar contains the following buttons:

  • Import INP: allows you to import an inp file. For more details, see the section Dialog Import INP.

  • Boundary conditions manager: allows you manage the boundary conditions. For more details, see the section Dialog Boundary conditions manager.

  • Create boundary condition: allows you create the boundary conditions. For more details, see the section Dialog Create boundary conditions.

  • Non visual objects manager: allows you manage the non visual objects of the project. For more details, see the section Dialog Non visual objects.

  • Bridge actions: allows you add or edit a bridge. For more details, see the section Dialog Bridge Actions.

7.2. Toolbar Utilities

This toolbar contains the following buttons:

  • Options: it allows you to manage all SWMM and Iber options in one place. For more details, see the section Dialog Go2Iber options.

  • Generate INP: it allows you to export to an inp file the network. For more details, see the section Dialog Generate INP.

  • Mesh manager: it allows you to manage the meshs of the project. For more details, see the section Dialog Mesh manager.

  • Execute model: it allows you to execute the model to simulate the network. For more details, see the section Dialog Execute model.

  • Results: it contains many tools for working with the simulation results. For more details, see the section Dialog Results.

  • Check project: it allows you to check the project in order to find errors. For more details, see the section Dialog Check project.
